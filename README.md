
# SOPP+IA Sofía | Predictor educativo de DDC

Aplicación académica desarrollada en Python y Streamlit para explorar factores de riesgo de Displasia del Desarrollo de Cadera y estimar una probabilidad educativa mediante Machine Learning.

## Archivos principales

- `app.py`: aplicación principal en Streamlit.
- `dataset_sintetico_ddc.csv`: dataset sintético usado para entrenamiento.
- `requirements.txt`: librerías necesarias.
- `assets/soppia_sofia_icon.png`: ícono visual de SOPP+IA Sofía.

## Advertencia

Este proyecto es académico y educativo. No reemplaza el criterio médico, el examen físico ni los estudios de imagen.
