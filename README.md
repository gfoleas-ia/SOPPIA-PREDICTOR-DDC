# SOPP+IA Sofía | Predictor educativo de DDC con método de Graf

Aplicación académica desarrollada en Python y Streamlit para entrenamiento de padres, madres y cuidadores sobre la Displasia del Desarrollo de Cadera (DDC). Usa un dataset sintético auditado con variables clínicas, radiográficas y ecográficas del método de Graf para concientizar sobre diagnóstico temprano y tratamiento oportuno.

## Archivos principales

- `app.py`: aplicación principal en Streamlit.
- `dataset_sintetico_ddc_graf_auditado_v2.csv`: dataset sintético auditado con 1000 pacientes y 26 variables.
- `diccionario_variables_ddc_graf_auditado_v2.txt`: diccionario de variables del dataset.
- `requirements.txt`: librerías necesarias.
- `assets/soppia_sofia_icon.png`: ícono visual de SOPP+IA Sofía.

## Uso

```bash
streamlit run app.py
```

## Advertencia

Este proyecto es académico, educativo y demostrativo. No reemplaza el criterio médico, el examen físico, la ecografía, la radiografía ni la evaluación del médico especialista en Ortopedia Pediátrica.
